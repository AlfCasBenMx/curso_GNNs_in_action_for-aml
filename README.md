# 🌐 Curso: Graph Neural Networks in Action para AML

Materiales de estudio del libro **"Graph Neural Networks in Action"** (Keita Broadwater, Manning) aplicados a **Anti-Money Laundering**.

## 📚 Notebooks del Curso

| # | Tema | Notebook |
|---|------|----------|
| 0 | Fundamentos de Redes Neuronales | `0_Neural_Networks_Fundamentals.ipynb` |
| 1 | Graph Convolutional Networks (GCN) | `1_Graph_Convolutional_Networks.ipynb` |
| 2 | Graph Attention Networks (GAT) | `2_Graph_Attention_Network.ipynb` |
| 2.5 | Node2Vec - Graph Embeddings | `2_5_Node2Vec_Paso_a_Paso.ipynb` |
| 3 | GraphSAGE | `3_GraphSAGE.ipynb` |
| 4 | Graph Isomorphism Network (GIN) | `4_Graph_Isomorphism_Network.ipynb` |

Los notebooks están en `graph-neural-network-course/`. Los resúmenes HTML de cada capítulo en `capitulos/` y los datasets en `datos/`.

## 🛠️ Requisitos

Python 3.8+ · PyTorch · PyTorch Geometric · NetworkX
