📐 La Fórmula de R²

Todo se reduce a comparar DOS errores

R cuadrado se calcula así:

R² = 1 − SS_res / SS_tot
SS_res = Σ(yᵢ − ŷᵢ)²
Error de TU MODELO
Suma de las diferencias al cuadrado entre los valores reales y las predicciones de tu modelo
SS_tot = Σ(yᵢ − ȳ)²
Error de LA MEDIA
Suma de las diferencias al cuadrado entre los valores reales y la media (el "modelo tonto")
Interpretación
Si SS_res > SS_tot, la fracción es > 1, y por tanto R² < 0.
Significa: tu modelo se equivoca MÁS que la media.

🎯 ¿Por qué la media es tan difícil de superar?

La media tiene un "superpoder" matemático

01

La media MINIMIZA el error cuadrático

De todos los valores constantes que podrías elegir, la media es el que produce el menor error cuadrático total. Esto es un hecho matemático demostrado.

02

Nunca se va a los extremos

La media siempre está "en el centro" de los datos. No comete errores gigantes con ningún dato individual — los errores se distribuyen uniformemente.

03

Para ganarle, necesitas información ÚTIL

Un modelo que usa features irrelevantes o aprende ruido no tiene información útil. Sin información real, cualquier intento de "ser inteligente" solo añade error.

🔍 ¿Por qué ocurre R² negativo?

Diagnóstico: las causas más comunes y cómo solucionarlas

📈
Overfitting
El modelo memoriza el train set pero falla en test. Aprende ruido en vez de patrones reales.
✅ Regularización (L1, L2), Cross-validation, menos features
🗑️
Features Irrelevantes
Variables que no tienen relación real con el target. El modelo intenta usarlas y genera ruido.
✅ Feature selection, análisis de correlación, PCA
⚙️
Modelo Inadecuado
Usar un modelo lineal cuando la relación es no-lineal, o viceversa.
✅ Probar otros algoritmos: árboles, SVM, redes neuronales
📊
Datos Problemáticos
Outliers extremos, datos mal etiquetados, o escala incorrecta de variables.
✅ Limpieza de datos, normalización, detección de outliers
📉
Pocos Datos
Con muy pocos datos no hay suficiente información para que el modelo aprenda patrones reales.
✅ Recolectar más datos, data augmentation
🔀
Data Leakage inverso
Train y test no representan la misma distribución, o el split se hizo de forma incorrecta.
✅ Revisar el split, stratified sampling, temporal split

📌 Conclusión

Un R² negativo es una señal de alarma: tu modelo literalmente añade ruido en vez de encontrar patrones. La media gana porque es el punto de equilibrio matemático — no necesita información y nunca se equivoca de forma catastrófica.

Para superar a la media, tu modelo necesita información genuinamente útil sobre la relación entre las variables.
