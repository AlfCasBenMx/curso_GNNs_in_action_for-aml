Todo empieza con la neurona artificial, inspirada en las neuronas biológicas del cerebro. Es una función matemática que:

  • Recibe múltiples entradas (números)
  • Multiplica cada entrada por un peso (importancia)
  • Suma todo y añade un bias (ajuste fino)
  • Aplica una función de activación
  • Produce una salida (predicción)
🧠
Analogía: La neurona como un juez aprobando préstamos

Imagina un juez que decide si aprobar un préstamo. Recibe datos (ingresos=$5000, historial crediticio=750, deudas=$10000). Cada dato tiene un "peso" en su decisión:

  • Ingresos: peso=+0.8 (muy importante)
  • Historial: peso=+0.5 (importante)
  • Deudas: peso=-0.6 (penaliza)

Calcula: (0.8×5000) + (0.5×750) + (-0.6×10000) + bias = ¿Aprobar o rechazar?

Anatomía de una Neurona

📐 La Fórmula
ŷ = σ( w₁·x₁ + w₂·x₂ + w₃·x₃ + b )
● x = entradas ● w = pesos ● b = bias ● σ = activación

🔍 Desglosando cada componente:

1. Entradas (x): Los datos crudos
Ej: edad=25, salario=50000
2. Pesos (w): Importancia de cada entrada
Valores ajustables (se aprenden)
3. Bias (b): Ajuste fino/offset
Desplaza el umbral de activación
4. Activación (σ): Transforma la suma
Añade no-linealidad

📋 Código para copiar y pegar: una neurona en Python

Asumiendo que ya copiaste y ejecutaste el Paso 0 (imports), este bloque muestra cómo calcular la salida de una neurona con 3 entradas, pesos y bias, y luego pasarla por una función de activación sigmoide. 

import math

      # Entradas (x)
      x1, x2, x3 = 0.8, 0.5, 0.2  # por ejemplo: normalizadas en [0, 1]

      # Pesos (w) y bias (b)
      w1, w2, w3 = 0.9, 0.4, -0.6
      b = 0.1

      def sigmoid(z: float) -> float:
        return 1 / (1 + math.exp(-z))

      # Suma ponderada (logit)
      z = w1 * x1 + w2 * x2 + w3 * x3 + b

      # Salida de la neurona después de la activación
      y_hat = sigmoid(z)

      print(f"z = {z:.3f}")
      print(f"σ(z) = {y_hat:.3f}")

Puedes cambiar los valores de x1, x2, x3 y de los pesos w1, w2, w3 para ver cómo afecta a la salida de la neurona.

💡
¡Punto clave! Los pesos (w) y bias (b) son los únicos parámetros que la red APRENDE. Al inicio se inicializan aleatoriamente (ej: valores pequeños cerca de 0), y durante el entrenamiento se van ajustando mediante backpropagation para minimizar el error.
