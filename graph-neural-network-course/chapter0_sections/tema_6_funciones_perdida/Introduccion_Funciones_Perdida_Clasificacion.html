🔧 Las Funciones Principales

🎲

BCE - Binary Cross-Entropy

Para 2 clases (binaria)

BCE = -(1/n) · Σ[y·log(p) + (1-y)·log(1-p)]

✅ Ventaja: Funciona con probabilidades

⚠️ Importante: Requiere sigmoid antes

🎯 Usa para: Spam detection, clasificación simple

⭐

BCE with Logits

MÁS ESTABLE (Recomendado)

Aplica Sigmoid + BCE automáticamente

✅ Ventaja: Más estable numéricamente

✅ Ventaja: Combina ambas operaciones

🎯 Usa para: Producción, siempre que puedas

💡 Esta es la versión PRO

🌈

Cross-Entropy Loss

Para múltiples clases

CE = -(1/n) · Σ log(softmax(x)_y)

✅ Ventaja: Incluye softmax automático

✅ Ventaja: Funciona con N clases

🎯 Usa para: ImageNet, MNIST, multi-clase

⚖️

Weighted BCE

Para clases desbalanceadas

pos_weight = #negativas / #positivas BCEWithLogitsLoss(pos_weight=w)

✅ Ventaja: Maneja desbalanceo

✅ Ventaja: Crucial para AML/fraude

🎯 Usa para: Datos desbalanceados (99% vs 1%)

⚠️ Errores Comunes a Evitar

  • NO aplicar sigmoid/softmax manualmente si usas BCEWithLogitsLoss o CrossEntropyLoss
  • NO usar BCE simple - usa BCEWithLogitsLoss (más estable)
  • NO ignorar el desbalanceo - usa pos_weight para datos desbalanceados
  • NO confundir etiquetas: BCE usa 0/1, CrossEntropy usa índices 0,1,2...

📊 Tabla Comparativa Rápida

Función PyTorch Clases ¿Aplica Sigmoid/Softmax?
BCE nn.BCELoss() 2 (binaria) ❌ No (debes hacerlo tú)
BCE with Logits ⭐ nn.BCEWithLogitsLoss() 2 (binaria) ✅ Sí (automático)
Cross-Entropy nn.CrossEntropyLoss() 2+ (multi-clase) ✅ Sí (automático)

💼 Casos de Uso en el Mundo Real

  • Finanzas - AML/Fraude: Weighted BCE (98% legítimas, 2% fraude)
  • Email Marketing: BCE with Logits (spam vs no spam)
  • Computer Vision: Cross-Entropy (clasificar 1000 objetos)
  • Medicina: Weighted BCE (diagnóstico de enfermedades raras)
  • Retail: Cross-Entropy (categorizar productos)
  • Cybersecurity: Weighted BCE (detección de ataques)
PRO TIP: En AML y fraude, el costo de NO detectar un caso positivo (False Negative) es MUCHO mayor que tener falsas alarmas (False Positives). ¡Usa pos_weight para reflejarlo!
