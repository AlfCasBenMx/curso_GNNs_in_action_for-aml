📚 ¿Qué es una Función de Pérdida?

Una función de pérdida (loss function) mide qué tan lejos están nuestras predicciones de los valores reales. Es la brújula que guía al modelo durante el entrenamiento.


Analogía: Si estás aprendiendo a tirar dardos, la función de pérdida es como medir la distancia entre donde diste y el centro del blanco. ¡Mientras más pequeña, mejor!

🔧 Las 4 Funciones Principales

📐

MSE - Mean Squared Error

Error Cuadrático Medio

MSE = (1/n) · Σ(y - ŷ)²

✅ Ventaja: Penaliza mucho los errores grandes

❌ Desventaja: Sensible a outliers

🎯 Usa cuando: Quieres castigar errores grandes (ej: precios de casas)

📏

MAE - Mean Absolute Error

Error Absoluto Medio

MAE = (1/n) · Σ|y - ŷ|

✅ Ventaja: Fácil de interpretar, robusto

❌ Desventaja: Trata todos los errores igual

🎯 Usa cuando: Tienes outliers o quieres interpretabilidad directa

⚖️

Smooth L1 Loss (Huber)

Lo Mejor de Ambos Mundos

Smooth L1 = { 0.5 · error² si |error| < 1 |error| - 0.5 si |error| ≥ 1 } Promedio de todas las losses

✅ Ventaja: Combina MSE + MAE inteligentemente

✅ Ventaja: Suave cerca de 0, lineal para errores grandes

🎯 Usa cuando: Object detection, vision computacional

📊

RMSE - Root Mean Squared Error

MSE en Unidades Originales

RMSE = √MSE = √[(1/n) · Σ(y - ŷ)²]

✅ Ventaja: Mismas unidades que los datos

✅ Ventaja: Fácil de interpretar

🎯 Usa cuando: Necesitas comunicar resultados (ej: ±5k de error)

📊 Comparación Rápida

Función PyTorch Mejor Para Sensibilidad a Outliers
MSE nn.MSELoss() Predicción de precios, temperaturas ⚠️ Alta
MAE nn.L1Loss() Datos con outliers, estimaciones ✅ Baja
Smooth L1 nn.SmoothL1Loss() Object detection, bounding boxes ✅ Media
RMSE torch.sqrt(nn.MSELoss()) Reportes, interpretación fácil ⚠️ Alta (como MSE)

💼 Casos de Uso en el Mundo Real

  • Bienes Raíces: Predicción de precios de casas (MSE)
  • Finanzas: Predicción de precios de acciones (MAE o Smooth L1)
  • Retail: Forecasting de ventas (MSE)
  • Clima: Predicción de temperaturas (MAE)
  • Computer Vision: Detection de objetos (Smooth L1)
  • IoT: Predicción de consumo energético (MSE)
