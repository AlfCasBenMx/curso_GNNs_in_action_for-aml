Regresión, binaria y multiclase

📏
MSE (batch=3):
y = [2, 0, 1], ŷ = [1, 1, 2]
Errores: (2-1)²=1, (0-1)²=1, (1-2)²=1 → Σ=3
MSE = 3/3 = 1.0
CrossEntropy multiclase (4 clases):
Logits z=[2,1,-1,0] → Softmax=[0.62,0.23,0.08,0.17]
Clase real = 1 → CE = -log(0.23) = 1.47
Si la red mejora a 0.60 para la clase real → CE baja a 0.51.
BCEWithLogits (binaria robusta):
Usa log-sum-exp estable: loss = log(1+exp(-y·z)) donde y∈{-1,1}.
Ej: z=2.0, y=1 → loss ≈ 0.126; z=-2.0, y=1 → loss ≈ 2.126.
Tip: preferir BCEWithLogitsLoss sobre Sigmoid+BCELoss por estabilidad.
