El Famoso Problema XOR

Ejemplo clásico de no-linealidad

✅ Al terminar esta sección deberías poder:

  • Explicar por qué XOR no es linealmente separable en el plano.
  • Describir cómo una capa oculta puede "doblar" el espacio para hacer separable XOR.
  • Generar o interpretar un dataset XOR sintético y ver que ninguna línea recta separa las clases.

XOR es el ejemplo clásico de problema NO linealmente separable. Minsky y Papert demostraron en 1969 que un perceptrón simple NO puede resolverlo:

x₁ x₂ XOR
000
011
101
110

Intuición: XOR = 1 cuando
las entradas son diferentes

Los puntos (0,1) y (1,0) deben estar juntos (clase 1), pero están en esquinas opuestas.

Los puntos (0,0) y (1,1) deben estar juntos (clase 0), pero también están en esquinas opuestas.

¡No hay línea recta que pueda separarlos!

💡 Solución: Agregar una capa oculta que "transforme" el espacio de R² a una representación donde los puntos SÍ sean separables.

📐 ¿Cómo se generan los puntos?
1️⃣ Generación base
x₁, x₂ ~ Uniform(0, 1)
# Puntos uniformes en [0,1]²
2️⃣ Asignación XOR
y = (x₁ > 0.5) ⊕ (x₂ > 0.5)
# ⊕ = XOR lógico
3️⃣ Añadir ruido
x₁' = x₁ + ε, ε ~ N(0, σ²)
# Ruido gaussiano
⚠️ El problema: Ninguna línea w₁x₁ + w₂x₂ + b = 0 puede separar las clases porque los puntos de la misma clase están en cuadrantes opuestos (diagonal). Se necesita una transformación no lineal.
🎯
Insight Importante: Conocemos la Función Generadora

En este ejemplo "de juguete", nosotros definimos la regla que genera los datos:

f*(x₁, x₂) = (x₁ > 0.5) ⊕ (x₂ > 0.5) ← Esta es la "verdad" (ground truth)

La red neuronal intenta aproximar f* sin conocerla explícitamente, solo viendo ejemplos (x, y). En problemas reales no conocemos f* — pero aquí podemos verificar si la red la aprendió correctamente.

🔮 La Magia de las Capas Ocultas: "Doblar" el Espacio

La capa oculta transforma los puntos a un nuevo espacio donde SÍ son linealmente separables. Es como "doblar" una hoja de papel para que puntos lejanos queden juntos.

📋 Código para copiar y pegar: resolver XOR con una capa oculta

A partir de los imports del Paso 0, este ejemplo crea el dataset XOR clásico de 4 puntos y entrena una red con una capa oculta para aprender la función XOR. 

# Datos XOR (4 puntos)
X = torch.tensor([
    [0., 0.],
    [0., 1.],
    [1., 0.],
    [1., 1.],
])
y = torch.tensor([[0.], [1.], [1.], [0.]])  # XOR

print("x1 x2 | y")
for x_i, y_i in zip(X, y):
    print(int(x_i[0].item()), int(x_i[1].item()), " | ", int(y_i.item()))

# Red con 1 capa oculta de 4 neuronas
modelo_xor = nn.Sequential(
    nn.Linear(2, 4),
    nn.ReLU(),
    nn.Linear(4, 1),
    nn.Sigmoid()
)

criterio = nn.BCELoss()
opt = torch.optim.SGD(modelo_xor.parameters(), lr=0.1)

for epoca in range(5000):
    y_pred = modelo_xor(X)
    loss = criterio(y_pred, y)

    opt.zero_grad()
    loss.backward()
    opt.step()

    if epoca % 1000 == 0:
        print(f"Epoch {epoca} | Loss = {loss.item():.4f}")

print("\nPredicciones finales:")
with torch.no_grad():
    probs = modelo_xor(X)
    for x_i, p in zip(X, probs):
        pred = 1 if p.item() >= 0.5 else 0
        print(f"({int(x_i[0].item())}, {int(x_i[1].item())}) -> σ(z)={p.item():.2f}, pred={pred}")
