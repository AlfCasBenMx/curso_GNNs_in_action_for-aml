Backprop en Mini-Red

2 capas, ReLU intermedia

🧾
Setup:
x=1.0
Capa 1: z1 = w1·x + b1 = 0.5·1 + 0.0 = 0.5 → a1 = ReLU(z1)=0.5
Capa 2: z2 = w2·a1 + b2 = 2.0·0.5 + 0.0 = 1.0 → ŷ = z2 (regresión)
y_real = 0.2; Loss = (ŷ - y)² = (1.0-0.2)² = 0.64
Gradientes:
∂Loss/∂ŷ = 2(ŷ - y) = 1.6
∂ŷ/∂w2 = a1 = 0.5 → ∂Loss/∂w2 = 1.6 × 0.5 = 0.8
∂ŷ/∂a1 = w2 = 2.0 → ∂Loss/∂a1 = 1.6 × 2.0 = 3.2
ReLU'(z1)=1 (porque z1>0) → ∂Loss/∂z1 = 3.2
∂z1/∂w1 = x = 1.0 → ∂Loss/∂w1 = 3.2 × 1 = 3.2
∂Loss/∂b1 = 3.2, ∂Loss/∂b2 = 1.6
Update (lr = 0.1):
w2_new = 2.0 - 0.1×0.8 = 1.92
w1_new = 0.5 - 0.1×3.2 = 0.18
b2_new = -0.16; b1_new = -0.32
Idea clave: la regla de la cadena pasa el error hacia atrás multiplicando gradientes locales.
