¿Qué es una Capa Oculta?

El ingrediente secreto de las redes neuronales

✅ Al terminar esta sección deberías poder:

  • Explicar en palabras qué hace una capa oculta entre la entrada y la salida.
  • Distinguir entre "más neuronas" y "más capas" y su efecto en la capacidad del modelo.
  • Leer un diagrama simple de MLP (2 → [h] → 1) e identificar entrada, capas ocultas y salida.

Ya vimos que una neurona sola solo puede dibujar líneas rectas. Para resolver problemas como XOR o círculos, necesitamos capas ocultas. Pero, ¿qué son exactamente?

🏭
Analogía: La fábrica de procesamiento

Imagina que quieres decidir si una foto muestra un gato. El proceso tiene etapas:

  • Entrada: Píxeles crudos (números)
  • Capa Oculta 1: Detecta bordes y líneas básicas
  • Capa Oculta 2: Combina bordes → detecta ojos, orejas, bigotes
  • Capa Oculta 3: Combina partes → detecta cara de gato
  • Salida: ¿Es gato? Sí/No

Cada capa oculta extrae representaciones más abstractas. ¡Por eso se llama "deep learning" — aprendizaje profundo!

🔗 Anatomía de una Red con Capas Ocultas

¿Por qué se llaman "ocultas"?

Porque no las vemos directamente. Solo vemos la entrada (datos) y la salida (predicción). Las capas intermedias "procesan" internamente — sus valores son representaciones aprendidas automáticamente.

🧮 ¿Qué hace cada neurona oculta?

Cada neurona calcula: σ(w₁x₁ + w₂x₂ + b). Con diferentes pesos, cada neurona "busca" un patrón diferente en los datos.

🔗 ¿Por qué combinarlas?

La siguiente capa combina los patrones detectados. Así, patrones simples (líneas) forman patrones complejos (círculos, formas).

📊 ¿Cuántas capas y neuronas?

Más neuronas: más patrones detectables
Más capas: patrones más abstractos
⚠️ Demasiadas → riesgo de overfitting

✨ El poder de la profundidad

Redes con múltiples capas pueden aproximar cualquier función continua (Teorema de Aproximación Universal).

Resumen: Una capa oculta es un grupo de neuronas entre la entrada y la salida que transforman los datos en representaciones más útiles. Cada capa adicional permite detectar patrones más abstractos. En la siguiente sección veremos esto en acción con el problema del círculo.

📋 Código para copiar y pegar: red con una capa oculta

Partiendo de los imports del Paso 0, este ejemplo define una red muy simple con una capa oculta y activación ReLU. Es el bloque básico que usarás luego para problemas como XOR o el círculo. 

class MLP(nn.Module):
    def __init__(self, in_dim=2, hidden_dim=4, out_dim=1):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(in_dim, hidden_dim),
            nn.ReLU(),   # capa oculta con activación no lineal
            nn.Linear(hidden_dim, out_dim)
        )

    def forward(self, x):
        return self.net(x)


modelo = MLP()
print(modelo)

# Ejemplo de forward con un solo punto (x1, x2)
ejemplo = torch.tensor([[0.3, -0.5]])  # shape [batch=1, features=2]
salida = modelo(ejemplo)
print("Salida cruda (logit):", salida.item())
