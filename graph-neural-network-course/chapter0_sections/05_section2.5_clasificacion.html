De Predecir Números a Clasificar

Regresión vs Clasificación

✅ Al terminar esta sección deberías poder:

  • Distinguir claramente entre regresión y clasificación binaria.
  • Interpretar σ(z) como probabilidad p(y = 1 | x) en un problema 0/1.
  • Saber cuándo usar Sigmoid + Binary Cross-Entropy para clasificar.

Hasta ahora hemos visto cómo predecir valores continuos (precio de una casa). Pero en muchos problemas queremos clasificar: ¿Es spam o no spam? ¿Es un gato o un perro? ¿Aprobar o rechazar?

Predecir una categoría
  • Email: Spam / No Spam
  • Imagen: Gato / Perro
  • Préstamo: Aprobar / Rechazar
Salida: 0 o 1 (binaria)

🔮 ¿Cómo convertimos una neurona en clasificador?

Usamos la función Sigmoid (σ) que "aplasta" cualquier número al rango [0, 1]:

✨ La magia del Sigmoid:

  • Entrada muy negativa → salida ≈ 0
  • Entrada muy positiva → salida ≈ 1
  • Entrada = 0 → salida = 0.5

📊 Interpretamos la salida como probabilidad:
σ(z) > 0.5 → Clase 1 | σ(z) ≤ 0.5 → Clase 0

z = 0.0

¿Qué está haciendo realmente la neurona?
Está dibujando una línea (o plano) divisoria en el espacio de datos. Los puntos de un lado → Clase 0, los del otro lado → Clase 1. Esto se llama frontera de decisión.

📋 Código para copiar y pegar: clasificación binaria con Sigmoid

Partiendo de los imports del Paso 0, este ejemplo crea un dataset sencillo de casas etiquetadas como cara (1) o barata (0) según un umbral de precio, y entrena una neurona con activación Sigmoid para aproximar esa clasificación. 

# Datos de ejemplo: m² y etiqueta cara (1) / barata (0)
x_m2 = torch.tensor([[40.], [60.], [80.], [100.], [120.], [150.]] )
precios = 2.5 * x_m2 + 50  # modelo "real" oculto

umbral_precio = 260  # >= 260k -> cara (1), si no barata (0)
labels = (precios >= umbral_precio).float()

print("m²  precio(k)  etiqueta")
for x, p, y in zip(x_m2, precios, labels):
    print(f"{x.item():4.0f}   {p.item():7.1f}     {int(y.item())}")

# Modelo logístico: 1 neurona con Sigmoid
modelo_clf = nn.Sequential(
    nn.Linear(1, 1),
    nn.Sigmoid()
)

criterio = nn.BCELoss()  # Binary Cross-Entropy
opt = torch.optim.SGD(modelo_clf.parameters(), lr=1e-2)

for epoca in range(2000):
    y_pred = modelo_clf(x_m2)
    loss = criterio(y_pred, labels)

    opt.zero_grad()
    loss.backward()
    opt.step()

print("\nP(x es 'cara' | m²):")
with torch.no_grad():
    probs = modelo_clf(x_m2)
    for x, p, prob in zip(x_m2, precios, probs):
        decision = 1 if prob.item() >= 0.5 else 0
        print(f"{x.item():4.0f} m²  precio≈{p.item():6.1f}k  σ(z)={prob.item():.2f}  -> pred={decision}")
