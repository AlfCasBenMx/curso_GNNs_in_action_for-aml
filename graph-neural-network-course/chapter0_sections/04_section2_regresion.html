La regresión lineal es la red neuronal más simple: 1 neurona, sin función de activación. ¿Por qué? Porque la salida es directamente una combinación lineal de las entradas.

🏠
Ejemplo concreto: Predecir precio de casa

Queremos predecir el precio basándonos solo en los metros cuadrados. Es una relación simple:

  • Input (x): metros cuadrados
  • Output (ŷ): precio estimado
  • Modelo: ŷ = w×x + b (una línea recta)

La red aprende w (cuánto vale cada m²) y b (precio base). Por ejemplo: w=2500, b=50000 significa "$50k base + $2.5k por cada m²".

Entradas: 1 (máx 5)
📐 Fórmula:
ŷ = w₁x₁ + b

📋 Código para copiar y pegar: regresión lineal simple

Partiendo de que ya importaste las librerías (Paso 0), este bloque crea un dataset sintético de casas, ajusta una regresión lineal con PyTorch y dibuja la recta aprendida sobre los datos. 

# Datos: metros cuadrados -> precio de casa (en miles)
metros_cuadrados = torch.tensor([[30.], [50.], [70.], [80.], [100.], [120.], [150.]])
precios_reales = 2.5 * metros_cuadrados + 50 + torch.randn(7, 1) * 10

plt.figure(figsize=(6, 4))
plt.scatter(metros_cuadrados.numpy(), precios_reales.numpy(), s=60, c="blue", label="Datos reales")
plt.xlabel("Metros cuadrados")
plt.ylabel("Precio (miles $)")
plt.title("Precio de casas vs Metros cuadrados")
plt.grid(True, alpha=0.3)
plt.legend()
plt.show()

# Modelo lineal: 1 entrada -> 1 salida
modelo_lineal = nn.Linear(1, 1)

criterio = nn.MSELoss()
optimizador = torch.optim.SGD(modelo_lineal.parameters(), lr=1e-3)

for epoca in range(1000):
    pred = modelo_lineal(metros_cuadrados)
    loss = criterio(pred, precios_reales)

    optimizador.zero_grad()
    loss.backward()
    optimizador.step()

print("w aprendido:", modelo_lineal.weight.item())
print("b aprendido:", modelo_lineal.bias.item())

# Dibujar la recta aprendida
x_linea = torch.linspace(20, 160, 100).unsqueeze(1)
y_linea = modelo_lineal(x_linea).detach()

plt.figure(figsize=(6, 4))
plt.scatter(metros_cuadrados.numpy(), precios_reales.numpy(), s=60, c="blue", label="Datos reales")
plt.plot(x_linea.numpy(), y_linea.numpy(), c="red", label="Regresión lineal")
plt.xlabel("Metros cuadrados")
plt.ylabel("Precio (miles $)")
plt.title("Regresión lineal: y = wx + b")
plt.grid(True, alpha=0.3)
plt.legend()
plt.show()

💡
¡Observa la diferencia!

Con 1 dimensión: la predicción es una línea (ŷ = w₁x₁ + b).
Con 2 dimensiones: la predicción es un plano (ŷ = w₁x₁ + w₂x₂ + b).
Cada dimensión extra añade un nuevo peso que la red debe aprender.

🤔
Pero hay un problema...
La regresión lineal solo puede aprender líneas rectas. ¿Qué pasa si los datos forman una curva o un círculo?
